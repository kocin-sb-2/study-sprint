## About

Study Sprint is a web-based study platform covering:

- 🇸🇪 **Swedish Gymnasium**: Fysik 2, Kemi 2, Matematik 5
- 🌍 **IB Diploma Programme HL**: Physics, Chemistry, Mathematics AA

Features include section/topic toggles, progress tracking, in-page search, quiz mode, concept maps, and study logging.

## Tech Stack

- Static HTML/CSS/JS (no build step)
- Deployed on Vercel
- localStorage for client-side progress
- Mermaid.js for concept diagrams

## Running Locally

```bash
# Any static file server works:
npx serve .
# or
python3 -m http.server 8000
```

Then open `http://localhost:8000` in your browser.
